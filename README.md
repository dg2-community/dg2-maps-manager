# DG2 Community Maps Manager

Windows preview app for managing Defense Grid 2 community maps.

Browse, import, organize, and manage DG2 custom maps without the usual manual file handling.

[Download latest release](https://github.com/dg2-community/dg2-maps-manager/releases/latest/download/DG2-Community-Maps-Manager.zip)

![DG2 Community Maps Manager preview](assets/dg2-manager-preview.webp)

## Why this app exists

Defense Grid 2 community maps are still worth playing, but installing and managing them by hand can be awkward.

DG2 Community Maps Manager was created to make the process easier and keep everything organized in one place.

### Built-in map catalog

Browse known DG2 community maps from inside the app instead of searching through scattered download links. See which maps are already in your local library.

### Correct ZIP importing

Defense Grid 2 expects map folders to sit directly inside `Documents\DG2`, but Windows ZIP extraction can sometimes create an extra folder layer that breaks map installs.

The manager handles the correct folder structure when importing ZIPs from [dg2freemaps.com](https://www.dg2freemaps.com/category/forever-free-maps/) using either the Import button or drag and drop.

### Active map control

Having too many custom maps active at once can make Defense Grid 2 unstable on some systems.

The manager helps you maintain a larger local library while only keeping the maps you currently want to play active inside the DG2 folder.

### Map previews

Since custom-map thumbnails are no longer available in-game, the manager brings previews back into the workflow so maps are easier to recognize before launching the game.

### Lightweight and local

No installer, account, launcher service, or background processes.

The app manages your local map library and active DG2 folder.

## How to use

1. Download the ZIP.
2. Extract the full folder.
3. Start `DG2 Community Maps Manager.exe`.
